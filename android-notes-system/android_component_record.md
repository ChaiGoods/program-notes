# Android 组件记录类型分析

## 前言

在分析 Android 系统四大组件的交互过程中，包含很多组件状态的记录数据结构类型，了解这些类型有助于更深层次的理解 Android 系统组件的工作原理，系统与应用进程的交互过程，以及对四大组件管理服务 ActivityManagerService 的理解。

## ApplicationRecord

ApplocationRecord 不是组件的记录类型，它是应用程序进程的记录类型，它和其他组件记录类型的联系是不可分割的。
